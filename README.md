# 💅 Telegram-бот для мастера маникюра

Бот для онлайн-записи к мастеру маникюра с админ-панелью, inline-календарём,
автонапоминаниями и проверкой подписки на канал.

---

## 📁 Структура проекта

```
nail_bot/
├── bot.py                  # Точка входа — запуск бота
├── config.py               # Конфигурация (токен, ID, тексты)
├── database.py             # Работа с SQLite
├── calendar_kb.py          # Inline-календарь
├── states.py               # FSM-состояния
├── scheduler.py            # APScheduler — автонапоминания
├── requirements.txt        # Зависимости
├── README.md               # Документация
└── handlers/
    ├── __init__.py          # Регистрация роутеров
    ├── user.py              # Хэндлеры пользователя
    ├── admin.py             # Хэндлеры админ-панели
    └── subscription.py      # Проверка подписки на канал
```

---

## ⚙️ Настройка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка `config.py`

Откройте `config.py` и заполните:

| Параметр              | Описание                                        |
|-----------------------|-------------------------------------------------|
| `BOT_TOKEN`           | Токен бота (получить у [@BotFather](https://t.me/BotFather)) |
| `ADMIN_ID`            | Ваш Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot)) |
| `SCHEDULE_CHANNEL_ID` | ID канала для публикации расписания              |
| `CHANNEL_ID`          | ID канала для проверки подписки                  |
| `CHANNEL_LINK`        | Ссылка на канал для кнопки «Подписаться»         |

> **Важно:** Бот должен быть администратором в канале, чтобы проверять подписки
> и публиковать сообщения.

### 3. Запуск

```bash
python bot.py
```

## 🚀 Функционал

### Для клиентов:
- 📅 **Записаться** — выбор даты и времени через inline-календарь
- ❌ **Отменить запись** — освобождает слот
- 💰 **Прайсы** — отображает прайс-лист
- 📸 **Портфолио** — ссылка на Pinterest
- 🔔 **Автонапоминание** — за 24 часа до визита
- 📢 **Проверка подписки** — обязательная подписка на канал

### Для администратора (`/admin`):
- 📅 Добавление рабочих дней
- 🕐 Управление временными слотами (добавление / удаление)
- 📋 Просмотр расписания на выбранную дату
- 🚫 Полное закрытие дня (с уведомлением клиентов)
- ❌ Отмена записей клиентов (с уведомлением)

### Ограничения:
- Один пользователь — одна активная запись
- Расписание формируется на 1 месяц вперёд
- При отмене записи слот автоматически освобождается

## 🔧 Технологии

- **Python 3.10+**
- **aiogram 3.x** — асинхронный фреймворк Telegram Bot API
- **SQLite** — локальная база данных
- **APScheduler** — планировщик задач для напоминаний
- **FSM** — конечный автомат для управления состояниями диалога
